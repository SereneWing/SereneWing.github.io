# Skill 03: Add New Module or Feature

## Overview

This skill is used when the task is to add a new module or feature to the project.

## Trigger

Task is to add a new module or feature to the project.

## Action

- **Permission**: MUST NOT modify AI_START.md
- Follow Core Rules (Section 2) when making changes

## TODO List

### 3.1 Plan New Module
3.1.1 Read AI_GUIDE.md (project overview)  
3.1.2 Understand existing module structure  
3.1.3 Determine appropriate module location  
3.1.4 Choose English name for new module (rule 2.1.2)  

### 3.2 Study Existing Patterns
3.2.1 Read existing module AI_GUIDE.md for reference  
3.2.2 Check README.md format in similar modules  
3.2.3 Review CSS variables and styling patterns  

### 3.3 Create Module Structure
3.3.1 Create new module directory with English name  
3.3.2 Create AI_GUIDE.md for new module  
3.3.3 Create README.md for new module  
3.3.4 Create initial index.html if applicable  

### 3.4 Update Parent Documentation
3.4.1 Update root AI_GUIDE.md progress section  
3.4.2 Update parent module AI_GUIDE.md  
3.4.3 Update root README.md  
3.4.4 Ensure no Chinese in AI files (rule 2.1.1)  

### 3.5 Verify Changes
3.5.1 Verify all paths are correct  
3.5.2 Check file naming (English only)  
3.5.3 Ensure documentation is synchronized
