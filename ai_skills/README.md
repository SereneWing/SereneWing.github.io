# AI Skills Directory

This directory contains all AI skills that can be used for different scenarios.

## Structure

```
ai_skills/
├── README.md                        # This file - skills overview
├── skill-01-start-optimization.md  # Skill 01: Optimize AI_START
├── skill-02-project-inspection.md  # Skill 02: Full project inspection
├── skill-03-add-module.md          # Skill 03: Add new module
├── skill-04-modify-module.md       # Skill 04: Modify existing module
└── skill-05-other-tasks.md         # Skill 05: Other tasks
```

## Skills Overview

| # | Name | Use Case | Action |
|---|------|----------|--------|
| 01 | Start Optimization | Modify AI_START.md itself | MAY modify AI_START.md |
| 02 | Project Inspection | Full project self-check | MUST NOT modify files |
| 03 | Add Module | Create new module or feature | MUST NOT modify AI_START.md |
| 04 | Modify Module | Update existing functionality | MUST NOT modify AI_START.md |
| 05 | Other Tasks | Catch-all for miscellaneous tasks | MUST NOT modify AI_START.md |

## Usage

Each skill contains:
- **Trigger**: When to use this skill
- **Action**: What you may or may not do
- **TODO List**: Step-by-step guidance

## Rules

- Skills are referenced from AI_START.md Section 3
- Each skill is self-contained with its own TODO list
- Follow all Core Rules when executing any skill
