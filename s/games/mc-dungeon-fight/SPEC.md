# 方块战士 - 我的世界地下城风格

## 1. 概述

### 1.1 项目名称
方块战士 - 我的世界地下城风格

### 1.2 项目类型
交互式HTML5动作游戏

### 1.3 核心功能
玩家控制一个方块风格的孙悟空，点击地图移动，点击怪物进行连续攻击，击败怪物获得积分

### 1.4 目标用户
我的世界爱好者、西游记爱好者、休闲游戏玩家

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 玩家控制 | 点击位置方头人走过去 | 必须 |

## 4. 游戏流程

### 4.1 初始化流程

```
1. 页面加载完成
2. 初始化音频系统
3. 创建初始怪物 (2个)
4. 启动游戏循环
5. 启动怪物生成定时器
6. 自动播放背景音乐
```

### 4.2 玩家移动流程

```
1. 用户点击游戏区域
2. 如果点击的是怪物，进入攻击流程
3. 否则设置玩家目标位置
4. 每帧移动玩家向目标位置
5. 播放走路动画
6. 到达目标后停止
```

### 4.3 攻击流程

```
1. 玩家点击怪物
2. 计算距离，如果超出攻击范围则走过去
3. 进入攻击状态
4. 每300ms播放一次打击音效
5. 播放金箍棒挥舞动画
6. 怪物生命值-1
7. 怪物闪烁效果
8. 生命值归零时怪物消失，积分+1
```

### 4.4 怪物移动流程

```
1. 每帧随机选择移动方向
2. 以较慢速度移动
3. 碰到边界时反弹
4. 定时更换移动方向
```
