# 方块战士

一款我的世界地下城风格的HTML5动作游戏。

## 游戏简介

玩家控制一个方块风格的孙悟空，在草地和泥土的我的世界风格场景中，击败随机出现的僵尸和骷髅怪物。

## 游戏玩法

1. **移动**：点击游戏区域任意位置，角色会自动走过去
2. **攻击**：点击怪物，角色会走过去用金箍棒连续攻击
3. **击杀**：每个怪物需要攻击3次才能消灭
4. **计分**：每消灭一个怪物获得1分

## 操作说明

| 操作 | 说明 |
|------|------|
| 左键点击空白处 | 控制角色移动到该位置 |
| 左键点击怪物 | 攻击该怪物 |
| 右下角按钮 | 开关背景音乐 |

## 游戏特色

- 🎮 方块风格的玩家和怪物造型
- ⚔️ 金箍棒挥舞攻击动画
- 🎵 Minecraft风格的8-bit中国风背景音乐
- 💥 打击特效和得分弹出动画
- 👟 走路腿部动画效果
- 🧟 两种怪物：僵尸和骷髅
- ❤️ 怪物血条显示

## 技术实现

- 纯原生HTML5 + CSS3 + JavaScript
- Web Audio API 合成音效
- requestAnimationFrame 游戏循环
- CSS动画实现视觉效果

## 游戏参数

| 参数 | 值 |
|------|-----|
| 游戏区域 | 900 x 650 像素 |
| 玩家移动速度 | 4 像素/帧 |
| 怪物移动速度 | 1.2 像素/帧 |
| 怪物生命值 | 3 |
| 最大怪物数 | 6 |
| 怪物生成间隔 | 3 秒 |
