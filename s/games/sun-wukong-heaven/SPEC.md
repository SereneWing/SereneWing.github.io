# 南天门 - 规格说明书

## 1. 概述

### 1.1 项目名称
南天门 - 孙悟空大闹天宫

### 1.2 项目类型
交互式HTML5小游戏

### 1.3 核心功能
玩家在游戏中观看孙悟空在南天门前自由飞翔，点击孙悟空触发互动效果（音效、表情动画、乾坤圈光标）

### 1.4 目标用户
西游记爱好者、休闲游戏玩家

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 自动飞行 | 孙悟空在游戏区域内自动左右飞翔 | 必须 |
| 点击交互 | 点击孙悟空触发多种效果 | 必须 |
| 音效播放 | 点击时播放"哎呀"叫声 | 必须 |
| 表情气泡 | 显示"哎呀！"表情气泡 | 必须 |
| 受伤动画 | 孙悟空显示受伤表情动画 | 必须 |
| 乾坤圈光标 | 点击后3秒内鼠标变为乾坤圈样式 | 必须 |
| 位置跳跃 | 点击后孙悟空跳到新位置 | 必须 |
| 魔法光环 | 点击后显示旋转的魔法光环 | 必须 |
| 背景音乐 | 可开关的中国风背景音乐 | 可选 |

### 2.2 场景元素

| 元素 | 描述 | 动画 |
|------|------|------|
| 星星 | 随机分布在顶部区域 | 闪烁动画 |
| 云朵 | 飘浮在南天门周围 | 左右漂移 |
| 仙鹤 | 从左飞向右方 | 飞行动画 |
| 南天门 | 主体建筑 | 静止 |
| 灯笼 | 悬挂在门两侧 | 摇摆动画 |

## 4. 用户交互

### 4.1 点击孙悟空流程

```
1. 用户点击孙悟空
2. 检查冷却状态 (cooldown)
3. 播放"哎呀"音效
4. 添加 hurt 动画类 (500ms)
5. 在点击位置显示"哎呀！"气泡 (1秒后消失)
6. 显示3个旋转魔法光环 (2秒后消失)
7. 添加 qiankun-cursor 类 (3秒后移除)
8. 随机生成新的目标位置
9. 孙悟空飞向新位置
```

### 4.2 冷却机制

| 参数 | 值 |
|------|-----|
| 冷却时间 | 500ms |
| 目的 | 防止连续点击过快 |

### 4.3 乾坤圈光标持续时间
3秒后自动恢复默认光标
