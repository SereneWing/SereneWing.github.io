# 百科模块规格

## 1. 模块概述

### 1.1 模块名称
百科 (Knowledge Base)

### 1.2 模块路径
`content/wiki/`

### 1.3 模块描述
AI编码知识库，包含术语解释、最佳实践、设计模式、工具推荐等。可作为学习参考资料和模板示例库。

### 1.4 目标用户
AI编程爱好者、前端开发者、学习者

---

|------|------|

| 分类 | 说明 |
|------|------|

| 分类 | 说明 |
|------|------|

### 2.4 模板示例
| 分类 | 说明 |
|------|------|
| HTML模板 | 常用HTML结构模板 |
| CSS模板 | 常用CSS样式模板 |

### 2.5 资源推荐

## 3. 设计规范

### 3.1 颜色规范
遵循项目主规格 `SPEC.md` 中的颜色定义：
- 主色：`#FFD700` (金色)
- 背景色：`#0a0a2e` (深蓝)
- 文字色：`#ffffff` (白色)

### 3.2 布局
- 分类卡片式展示
- 支持分类筛选
- 搜索功能（计划中）
