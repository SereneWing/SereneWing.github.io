## 🎮 游戏概述

孙悟空在南天门前自由飞翔，玩家点击孙悟空触发互动效果。

### 核心玩法

1. 孙悟空自动在南天门前飞翔
2. 点击"背景音乐"按钮开启背景音乐
3. **点击孙悟空**触发效果：
   - 发出"哎呀！"叫声
   - 显示受伤表情动画
   - 鼠标变为乾坤圈（持续3秒）
   - 孙悟空跳跃到新位置
   - 周围出现魔法光环

### 特色功能

- 自由飞翔动画
- 精美场景元素（南天门、灯笼、云彩、星星、仙鹤）
- 动态效果（星星闪烁、云彩飘动、萤火虫、仙鹤飞舞）

### 文件结构

```

├── AI_RULE.md    # 本文件
├── index.html   # 游戏主文件
```

### 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- CSS 动画实现视觉效果
- Web Audio API 生成音效
- SVG Data URI 实现自定义光标
