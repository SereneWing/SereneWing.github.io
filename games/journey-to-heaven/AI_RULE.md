## 🎮 游戏概述

玩家控制孙悟空穿越天宫，躲避托塔李天王和哪吒的攻击，最终到达天宫获胜。

### 核心玩法

1. 点击"Start Game"开始
2. 孙悟空自动沿路径移动向天宫
3. **点击托塔李天王** → 发射宝塔攻击
4. **点击哪吒** → 哪吒飞向你攻击
5. 躲避所有攻击到达天宫即胜利

### 角色说明

| 角色 | 功能 |
|------|------|
| 托塔李天王 | 点击后发射宝塔攻击 |
| 哪吒 | 点击后飞行攻击 |
