# 孙悟空大闹天宫 - 南天门

一个精美的西游记主题互动小游戏。

## 🎮 游戏介绍

在游戏中，孙悟空（🐒）会在南天门前自由飞翔，玩家可以点击孙悟空触发有趣的互动效果。

## 🕹️ 如何游玩

1. 打开游戏即可看到孙悟空在南天门前飞来飞去
2. 点击 **"背景音乐"** 按钮开启大闹天宫风格背景音乐
3. **点击孙悟空** 触发以下效果：
   - 发出"哎呀！"的可爱叫声
   - 显示受伤表情动画
   - 鼠标光标变成金光闪闪的乾坤圈（持续3秒）
   - 孙悟空会跳到新位置继续飞翔
   - 周围出现魔法光环效果

## ✨ 特色功能

- **自由飞翔**：孙悟空会随机在南天门前的天空中自由飞行
- **精美场景**：包含南天门、灯笼、云彩、星星、仙鹤等元素
- **动态效果**：
  - 闪烁的星星
  - 飘动的云彩
  - 摇曳的灯笼
  - 飞舞的萤火虫
  - 飞过的仙鹤
- **互动反馈**：
  - 点击音效
  - 表情气泡
  - 魔法光环
  - 乾坤圈光标

## 🎨 视觉设计

- **主题色**：金色 (#FFD700) + 深紫色天空
- **风格**：中国古典神话风格
- **动画**：流畅的飞行、闪烁、飘动效果

## 🔊 音频

- 背景音乐：大闹天宫风格中国风音乐
- 点击音效："哎呀"叫声

## 🛠️ 技术实现

- 纯 HTML5 + CSS3 + JavaScript
- CSS 动画实现流畅视觉效果
- Web Audio API 生成音效
- SVG Data URI 实现自定义光标

## 💡 提示

- 点击孙悟空后，乾坤圈光标会持续3秒后自动恢复
- 可以多次点击孙悟空，每次都会有不同位置的跳跃
- 建议开启背景音乐获得更好的游戏体验
