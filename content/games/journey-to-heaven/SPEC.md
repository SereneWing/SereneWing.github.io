# 大闹天宫 - 规格说明书

## 1. 概述

### 1.1 项目名称
大闹天宫 - 闯关小游戏

### 1.2 项目类型
交互式HTML5闯关游戏

### 1.3 核心功能
玩家控制孙悟空沿曲折路径前进，躲避托塔李天王和哪吒的攻击，最终到达天宫获得胜利

### 1.4 目标用户
西游记爱好者、休闲游戏玩家

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 开始游戏 | 点击"Start Game"按钮开始 | 必须 |
| 自动寻路 | 孙悟空沿预设路径自动前进 | 必须 |
| 路径系统 | Z字形曲折路径从左下到右上 | 必须 |
| 托塔李天王攻击 | 点击后发射宝塔攻击 | 必须 |
| 哪吒攻击 | 点击后哪吒飞起攻击 | 必须 |
| 碰撞检测 | 检测攻击是否击中孙悟空 | 必须 |
| 受击反应 | 被击中后闪烁并重置位置 | 必须 |
| 胜利判定 | 到达天宫区域触发胜利 | 必须 |
| 重新开始 | 游戏结束后可重新开始 | 必须 |
| 背景音乐 | 可开关的背景音乐 | 可选 |

### 2.2 游戏角色

| 角色 | 位置 | 攻击方式 |
|------|------|----------|
| 孙悟空 🐵 | 路径起点 | 被动移动 |
| 托塔李天王 👨‍🦱 | (150, 420) | 点击发射宝塔 🗼 |
| 哪吒 🦁 | (600, 550) | 起飞向上攻击 |

### 2.3 目标

| 目标 | 位置 | 胜利条件 |
|------|------|----------|
| 天宫 🏯 | (780, 85) | 孙悟空距离 < 70px |

## 4. 游戏流程

### 4.1 游戏状态机

```
[初始状态]
    ↓ 点击 Start Game
[运行状态] ←→ [受击状态]
    ↓              ↓
[胜利状态]    [继续运行]
    ↓
[结束状态] → 点击 Play Again → [运行状态]
```

### 4.2 托塔李天王攻击流程

```
1. 用户点击托塔李天王
2. 检查游戏是否运行、塔是否正在飞
3. 创建宝塔元素在位置 (190, 450)
4. 宝塔向右飞行
5. 每帧检测碰撞
6. 若碰撞：移除宝塔，触发 wukongHit()
7. 若飞到最大距离：渐隐返回
```

### 4.3 哪吒攻击流程

```
1. 用户点击哪吒
2. 检查游戏是否运行、哪吒是否正在飞
3. 创建飞行哪吒，隐藏原哪吒
4. Phase 1: 哪吒向上飞行
5. 每帧检测碰撞
6. 若碰撞：隐藏飞行哪吒，触发 wukongHit()
7. 若到达顶部：进入 Phase 2
8. Phase 2: 哪吒返回原位置
```

### 4.4 受击处理流程

```
1. 显示"被击中"提示
2. 取消游戏循环
3. 孙悟空闪烁8次 (150ms间隔)
4. 重置孙悟空到起点
5. 恢复游戏循环
```

### 4.5 胜利流程

```
1. 每帧检测孙悟空与天宫距离
2. 距离 < 70px 时触发胜利
3. 取消游戏循环
4. 显示胜利消息和烟花效果
5. 显示"Play Again"按钮
```
