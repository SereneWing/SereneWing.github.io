# 在线工具模块规格

## 1. 模块概述

### 1.1 模块名称
在线工具 (Online Tools)

### 1.2 模块路径
`content/tools/`

### 1.3 模块描述
提供实用的在线工具，帮助开发者提高工作效率。所有工具直接在浏览器端运行，无需服务器。

### 1.4 目标用户
前端开发者、AI编程爱好者

---

## 2. 工具列表

| 工具 | 状态 | 描述 |
|------|------|------|
| JSON格式化 | � 开发中 | JSON解析、验证和格式化 |
| 编解码器 | � 开发中 | Base64、URL编码等常用编解码 |
| 代码格式化 | 📋 计划中 | 代码美化/格式化工具 |

---

## 3. 设计规范

### 3.1 颜色规范
遵循项目主规格 `SPEC.md` 中的GitHub浅色风格定义：
- 背景色：`#ffffff` (白色)
- 主背景：`#f6f8fa` (浅灰)
- 边框色：`#d0d7de` (灰边框)
- 文字色：`#1f2328` (深灰)
- 次要文字：`#656d76` (中灰)
- 链接色：`#0969da` (蓝色)
- 强调色：`#0969da` (蓝色)

### 3.2 布局
- 工具卡片式展示
- 每个工具独立子目录
- 统一的输入输出区域样式

## 6. JSON格式化工具规格

### 6.1 功能特性
- JSON语法高亮
- 格式化/压缩切换
- 语法错误提示
- 复制到剪贴板

### 6.2 界面布局
```
┌─────────────────────────────────────┐
│  JSON格式化                    [复制]│
├─────────────────────────────────────┤
│  ┌─────────────────┐ ┌───────────┐ │
│  │     输入区域     │ │   输出区域  │ │
│  │                 │ │           │ │
│  │   (支持粘贴)    │ │  (高亮)   │ │
│  │                 │ │           │ │
│  └─────────────────┘ └───────────┘ │
│                                     │
│  [格式化] [压缩] [清空]              │
└─────────────────────────────────────┘
```
